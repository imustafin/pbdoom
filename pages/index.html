Installation

  1. Unzip the archive into the applications folder on your device
  2. Place your WAD file to applications/pbdoom folder
    WAD file name should be all lower case: (not DOOM2.WAD but doom2.wad)
  3. Your file structure should be like this:
    applications
├── pbdoom
│   ├── doom2.wad
│   ├── game.app
└── pbdoom.app
  4. Run the game using the pbdoom icon in the Applications menu

Device compatibility

PBDoom is known to be running on:

  • PocketBook X
  • PocketBook Touch HD 3
