Установка

На этой странице находятся инструкции для установки PBDoom на устройства PocketBook.

Последнюю версию игры можно скачать здесь:
Скачать PBDoom (v0.0.5)

  1. Скачайте архив с игрой

  2. Разархивируйте архив в папку applications на электронной книге

  3. Поместите ваши WAD файлы в папку applications/pbdoom. Имена WAD файлов должны быть написаны полностью маленькими буквами (не DOOM2.WAD, а doom2.wad).

  4. У вас должна получиться такая структура: 
    applications
├── pbdoom
│   ├── doom2.wad
│   ├── game.app
└── pbdoom.app
  5. Запустите игру, используя икноку pbdoom из меню приложений на электронной книге
