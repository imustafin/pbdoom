Installation

This page contains the instructions on how to install PBDoom on a PocketBook device.

The latest version of the game can be downloaded here:
Download PBDoom (v0.0.3)

  1. Download the game archive.

  2. Unzip the archive to the applications directory on the ereader.

  3. Place your WADs in the applications/pbdoom directory. WAD filenames should be all lower case (not DOOM2.WAD but doom2.wad).

  4. You should have this structure: 
    applications
├── pbdoom
│   ├── doom2.wad
│   ├── game.app
└── pbdoom.app
  5. Run the game using the pbdoom icon from the ereader applications menu
